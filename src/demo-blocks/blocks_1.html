This Demo #1 Animation uses ngAnimate and `addClass` callbacks to run custom TweenMax transitions.

The block directives must add or remove classes use `$animate.addClass`.
The `block` directives also use $q.all() to create the sequence chain.
The `.animation(".block", blockAnimations)` use TweenMax in the Services layer to specify the property transitions

Con: This animation is contained in the Directive layers of the app.
Con: Directives add classes and trigger Services to custom animation callback code [registered with .animate()]
Con: Complicated events are sent to update the Status table [using $emit()]
